## Keytool

Starter project for developing crypto-related key tools

```
Usage: index str_to_vec [options] <string>

Convert a base58 string to a u8 vec array

Arguments:
  string      string to translate

Options:
  -h, --help  display help for command
(base) Petes-M1:key-tool pete_o$ node index.js
Usage: index [options] [command]

Options:
  -h, --help           display help for command

Commands:
  str_to_vec <string>  Convert a base58 string to a u8 vec array
  help [command]       display help for command
```
